## 1. 大唐杯仿真笔记-5G网络开通与调测

**来源**：[wdkazzz / blog.csdn.net/wdkazzz/article/details/135636324](https://blog.csdn.net/wdkazzz/article/details/135636324)

### 知识点

- 仿真主线：**竞赛背景 → 勘站规划 → 网络部署 → 开通调测 → 业务验证**（文中还提到节能减排等目标）。
- **勘站规划**：射频规划三个方位角 **0°、120°、240°**；天线挂高按图中数字相加；下倾角按题目（电子+机械需相加）；宏站多用 **AAU**，NR 频段看设备名末几位、通道数/最大功率看参数表。
- **光纤与光模块**：IR 接口类型与模式决定选型——**单模 1310、多模 850**；**N78/N79** 常选 **1310、4×2LC-2LC（黑色）**；光模块常与光纤配套，IR 侧常见 **25G-1310**。
- **网络部署**强调：**各网元 IP 地址必须一致**；基站逻辑地址与核心网、业务服务器、BBU 等处 IP 对齐。
- **核心网网元表**（文中表格）：AMF、SMF、UDM、AUSF、UPF、PCF、NRF、NEF 等的中文名与职责说明（注册/连接/会话/策略/发现/能力开放等）。

### 仿真操作步骤（工程开通类）

1. **DTTP-6216**：按指引复制填写 IP，刷新接入后再做后续步骤。
2. **BBU**：板卡规划；设备接口与 **AAU 上下均需连线**；GPS 安装；辅材——**光纤与光模块配套**；**NG 口** 光纤 **1310-2LC-2LC**、光模块 **25G-1310**；**IR 口** 光纤 **1310 4×2LC-2LC**、光模块 **25G-1310**；供电 **直流 DC-48V**；小区频段按 AAU、注意 **PCI**；小区最大发射功率 **dBm**：**10·lg(mW)** 向下取整（例：240 W → 53 dBm）；NR 本地小区、传输规划（IP 由网关推算）；**版本升级**先空，邻区规划后据**告警日志**在 AAU/BBU 选版本；**NR 邻区**、异频载波、邻区关系按背景填；日志提取选告警→解析→版本信息处理；相关开关需打开。
3. **OMC 机柜**：传输参数中服务器地址可取 **OMC 网关地址 +1**（例：网关 `…::4` 则填 `…::5`）；**对端网络地址 = 对端掩码 ∧ OMC 服务器网关**；基站管理填基站逻辑地址；自治运维与**节能**相关。
4. **核心网规划**：签约号码；传输参数多为**网关+1**；路由计算方式同 OMC；网元功能配置；光纤 **850-2LC-2LC**、光模块 **25G-850**、供电 **交流 AC-220V**。
5. **业务服务器**：传输参数（业务服务器地址、UPF 地址）、路由、文件包（**10G/50G/100G** 等均可）。
6. **节能部署**：需同时改 **BBU 与 OMC**；按柱状图 max/min 填 **RRC 连接数、PRB 利用率** 等。
7. **业务验证**：FIP 运行次数、间隔、账号密码、服务器地址、下载线程、传输类型**二进制**、是否下载实体文档**否**、文件名一致、终端号码、网络模式 **SA**、小区 ID、**小区物理 ID** 等按题目与前面规划填写。

### 设备 / 工具信息

- **大唐仿真设备名**：**DTTP-6216**、**BBU**、**AAU**、**OMC 机柜**；光纤规格 **1310 / 850**、**2LC / 4×2LC-2LC**；光模块 **25G-1310**、**25G-850**；配电柜端子需先填电源至指示灯全绿再继续。

- 笔记仅供仿真辅助，**详细知识点需看官方网课**；基站规划部分**每年可能增新内容**，需结合官方与 B 站视频。
