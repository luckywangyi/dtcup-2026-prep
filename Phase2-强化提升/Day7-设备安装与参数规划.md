## 一、大唐设备安装规范回顾（Day1补充专题的强化）

### BBU安装 [真题多选]
- 竖装前方留**800mm**维护通道
- 支持并柜/并架安装
- 双排/多排时前后间隔≥800mm，同风向

### AAU安装
- 安装在铁塔/抱杆
- 方位角精度±5°，下倾角精度±0.5°
- IP65防水，-40°C~+55°C工作温度

### 供电与接地
- BBU/AAU均为**直流-48V**
- BBU接地线：**16平方**黄绿接地线
- 注意DCPD供电开关必须闭合

## 二、天线参数规划 [真题考点]

### 方位角规划原则 [真题-第九届B组]

正确的原则：
- 同站两扇区夹角**≥90°**
- 天线主瓣**朝向用户密集区域**
- 天线主瓣**不要朝向湖面/江面**

> **真题原题**：方位角规划原则不正确的是？→ 天线主瓣和道路夹角越小越好 [第九届B组]

### 下倾角调整 [真题判断-第十届A组]

| 场景 | 下倾角 | 原因 |
|------|--------|------|
| 过覆盖 | **增大**下倾角 | 缩小覆盖抑制干扰 |
| 灯下黑 | 增大下倾角 | 也能改善灯下黑 |
| 广覆盖 | 减小下倾角 | 扩大覆盖范围 |

> **真题判断**：过覆盖优化通常采用压下倾角，可以抑制过覆盖也可以缓解灯下黑 → **正确**

### 700MHz组网 [真题-第十届A组]

## 三、传输配置 [真题考点]

### 必配路由 [真题多选-第十届A组]

| 路由 | 用途 |
|------|------|
| 基站 → **AMF** | 控制面信令 |
| 基站 → **UPF** | 用户面数据 |
| 基站 → **OMC** | 网管管理 |

> 基站到业务服务器的路由**不是**必须配置的

## 四、频率规划

### NR-ARFCN计算

对于3000-24250MHz：
```
NR-ARFCN = (频率MHz - 3000) / 0.015 + 600000
```

### 信道栅格 [真题多选-第十届A组]

## 六、PCI规划复习

### PCI计算
```
PCI = 3 × N_ID_1 + N_ID_2
范围：0-1007
```

### PCI规划原则
- 同站三扇区**mod3不同**
- 相邻基站PCI不同
- mod30尽量不同（避免DMRS冲突）

> **真题判断**：PCI mod3/4相同对SS-SINR影响很小 → **正确** [第九届B组]
