## 二、多选题易错点汇总

### 陷阱1：RRC状态 vs NAS状态

- **RRC状态**：IDLE / INACTIVE / CONNECTED
- **NAS状态**：RM REGISTERED / RM DEREGISTERED
- 考试把NAS状态混入RRC选项中！

### 陷阱2：AMF vs SMF功能

| 功能 | 归属 |
|------|------|
| 注册管理 | AMF |
| 接入管理 | AMF |
| 移动性管理 | AMF |
| 会话建立/修改/删除 | **SMF** |
| UE IP地址分配 | **SMF** |
| 下行数据通知 | **SMF** |

### 陷阱3：PC5接口范围

- PC5覆盖：车-车(V2V)、车-RSU(UE级别)、车-人(V2P)
- **不含**：车-基站（走Uu口）

### 陷阱4：节能条件方向

- 进入节能：**低于低门限**
- 退出节能：**高于高门限**
- PRB利用率"高于低门限"是干扰项

### 陷阱5：设备供电

- BBU和AAU都是**直流-48V**
- 不是交流110V/220V
- 注意**负号**（-48V不是48V）

### 陷阱6：CCE聚合等级

- 合法：1/2/4/8/16
- **3不是**合法聚合等级

### 陷阱7："全选"型多选题

- 5G网络架构描述正确的 → ABCD
