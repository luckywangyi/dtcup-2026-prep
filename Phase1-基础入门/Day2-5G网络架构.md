## 二、CU/DU/AAU 三级分离

### 协议栈分割（必背）

```
CU = RRC + PDCP          ← 非实时处理
DU = RLC + MAC + 高PHY   ← 实时处理
AAU = 低PHY + RF         ← 射频处理
```

### CU-DU分割选项 [高频考点]

| Option | 分割点 | 传输带宽要求 | 时延要求 |
|--------|--------|-------------|---------|
| Option 1 | RRC/PDCP | 最低 | 最宽松 |
| Option 2 | PDCP/RLC | 低 | 宽松 |
| Option 5 | MAC上下 | 中 | 中 |
| **Option 8** | **PHY上下** | **最高** | **最严格** |

> **真题原题**：5G通信系统CU-DU分离方案中，哪个option对时延的要求最为严格？→ **D. Option 8** [第九届B组]
> **真题原题**：CU与DU划分方案中，哪个要求的传输带宽最高？→ **B. Option 8** [第九届B组]

### 接口定义

| 接口 | 连接 | 协议 |
|------|------|------|
| F1 | CU ↔ DU | F1AP |
| Xn | gNB ↔ gNB | XnAP |
| **NG** | gNB ↔ 5GC | NGAP |

> **真题原题**：5G基站和核心网通过哪个接口相连？→ **B. NG** [第九届A组]

## 三、5G核心网（5GC）

### 服务化架构（SBA）

5G核心网最大特点：服务化架构，网元通过服务化接口通信。

### 核心网元功能 [高频考点 — AMF vs SMF 必分清]

| 网元 | 功能 | 真题考法 |
|------|------|----------|
| **AMF** | **注册管理**、接入管理、移动性管理 | "AMF的功能是？→ 注册管理" |
| **SMF** | **会话建立/修改/删除**、UE IP分配、下行数据通知 | "会话的建立修改删除是？→ SMF" |
| **UPF** | 用户面转发、QoS执行 | 数据转发 |
| **NSSF** | 网络切片选择 | 切片相关 |
| **NRF** | 网元注册/发现 | 自动化管理 |

> **真题原题**：在5G网络架构中，以下哪项是AMF的功能？→ **A. 注册管理** [第九届A组]
> **真题原题**：NR系统中属于AMF网元的主要功能是？→ **D. 注册管理**（A/B/C均为SMF功能）[第十届A组]

> **易错点**：UE IP地址分配 = SMF功能（不是AMF！），会话建立修改删除 = SMF功能

### 接口（N系列）

| 接口 | 连接 | 说明 |
|------|------|------|
| N1 | UE ↔ AMF | NAS信令 |
| N2 | RAN ↔ AMF | 控制面 |
| N3 | RAN ↔ UPF | 用户面 |
| N4 | SMF ↔ UPF | 会话管理 |

## 六、软硬件解耦 [真题-第九届A组]

> **真题原题**：5G网络架构中，软件和硬件解耦主要通过什么技术实现？→ **D. NFV** [第九届A组]

| 技术 | 功能 |
|------|------|
| NFV | 网络功能虚拟化 → **软硬件解耦** |
| SDN | 软件定义网络 → 控制与转发分离 |
| MEC | 移动边缘计算 → 低时延 |
| 网络切片 | 逻辑网络隔离 |
