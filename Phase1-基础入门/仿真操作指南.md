## 一、5G网络开通与调测 [CSDN-wdkazzz]

### 完整流程

```
竞赛背景 → 勘站规划 → 网络部署 → 开通调测 → 业务验证
```

### 1. 勘站规划

**射频参数配置**：
| 参数 | 配置方法 |
|------|----------|
| 方位角 | 三扇区标准：**0°/120°/240°** |
| 天线挂高 | 根据题目给定数据相加 |
| 下倾角 | 电子下倾角 + 机械下倾角 |

> **真题原题**：方位角规划原则不正确的是？→ **C. 天线主瓣和道路夹角越小越好**（应该有适当角度）[第九届B组]
> 正确原则：同站两扇区夹角≥90°、不朝向湖面/江面、主瓣朝向用户密集区

**设备选型** [CSDN-wdkazzz]：
| 场景 | AAU型号 | MIMO | 频段识别 |
|------|---------|------|----------|
| 城区宏站 | 64T64R | Massive MIMO | 设备名称末位识别NR频段 |
| 郊区 | 32T32R | Massive MIMO | |
| 室内 | 4T4R/8T8R | 普通MIMO | |

**光纤光模块** [CSDN-wdkazzz]：
- 光纤：查IR接口类型，N78/N79选**1310单模**，4×2LC黑色
- 光模块：选**25G-1310**
- 电源：直流**DC -48V**

### 2. 网络部署

**设备组成**：
```
DTTP-6216(BBU) + AAU + OMC机柜 + 核心网 + 业务服务器
```

**IP地址配置关键**：
- 各设备IP必须一致
- 需配置三条路由：**基站→AMF、基站→UPF、基站→OMC** [真题多选]

### 3. 开通调测

- 检查板卡状态（HBPOD/HBPOF）
- 检查光模块方向（勿插反）
- 检查DCPD供电开关
- 配置小区参数（PCI、频点、带宽、SCS）

### 4. 业务验证

- UE附着测试
- Ping连通性测试
- 速率测试

## 二、5G信令流程仿真 [CSDN-JiangEZ]

### 七大信令流程

| 序号 | 流程 | 关键考点 |
|------|------|----------|
| 1 | 系统消息获取 | PSS/SSS→MIB→SIB1 |
| 2 | 随机接入+RRC连接 | 四步RACH：Msg1-4 |
| 3 | NAS鉴权加密 | Registration Request→Authentication |
| 4 | UE能力询问+AS加密 | SecurityModeCommand |
| 5 | RRC重配 | RRCReconfiguration |
| 6 | PDU Session建立 | DRB建立+QoS配置 |
| 7 | 测量切换 | A3事件→HandoverRequest |

### 仿真操作要点

**系统消息获取**：
1. 观察PSS/SSS检测 → 确认小区ID
2. 查看MIB → 确认SFN、SCS
3. 查看SIB1 → 确认RACH配置、TDD配置

**随机接入**：
1. Msg1：确认Preamble Index
2. Msg2(RAR)：查看**TA值、TC-RNTI、UL Grant**
3. Msg3：查看建立原因
4. Msg4：确认竞争解决

**基站到核心网第一条消息**：**Initial UE Message** [真题高频]

## 三、测量切换仿真

### 基站内切换
- 同一gNB内小区间切换
- 观察PCI变化

### Xn切换 [CSDN-JiangEZ]
```
源gNB → 目标gNB: Handover Request（第一条信令）
目标gNB → 源gNB: Handover Request Acknowledge
源gNB → UE: RRCReconfiguration
UE → 目标gNB: 随机接入 + RRCReconfigurationComplete
目标gNB → AMF: Path Switch Request
```

### NG切换
- 切换请求经AMF转发
- Handover Required (源→AMF) → Handover Request (AMF→目标)

## 四、射频单元参数规划

### 方位角配置
- 三扇区标准：**0°/120°/240°** 或 30°/150°/270°
- 从**正北**方向**顺时针**计算

### 下倾角
| 目标 | 调整 |
|------|------|
| 扩大覆盖 | 减小下倾角 |
| 减少干扰/过覆盖 | 增大下倾角 |

### 天线挂高
| 场景 | 推荐 |
|------|------|
| 密集城区 | 25-35m |
| 一般城区 | 30-45m |
| 郊区 | 35-55m |

## 六、仿真考试技巧 [CSDN-国一回忆录]

### 队友分工
- **每人主攻3-4个仿真题**
- 提前明确分工，比赛时各做各的
- 一人操作时另一人旁观检查

### 时间分配（30-45分钟）

| 阶段 | 时间 |
|------|------|
| 审题确认 | 2分钟 |
| 分工操作 | 20-30分钟 |
| 交叉检查 | 5-8分钟 |
| 提交 | 2-3分钟 |

### 注意事项
1. 使用**谷歌浏览器**
2. 选项顺序可能变化，**看内容不看位置**
3. IP地址配置注意**子网掩码正确性**
4. 提前登录测试环境
