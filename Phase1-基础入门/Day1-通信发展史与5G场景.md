# Day 1：通信发展史 + 5G三大场景

> 来源：真题统计 + CSDN 备赛帖 + 官方大纲

## 一、移动通信发展历程（30分钟速览）

| 代际 | 时期 | 核心技术 | 峰值速率 | 关键标准 |
|------|------|----------|----------|----------|
| 1G | 1980s | FDMA | 2.4kbps | AMPS/TACS |
| 2G | 1990s | TDMA/CDMA | 384kbps | GSM/IS-95 |
| 3G | 2000s | CDMA | 14.4Mbps | WCDMA/CDMA2000/**TD-SCDMA**(大唐) |
| 4G | 2010s | OFDMA+MIMO | 1Gbps | LTE/LTE-A |
| 5G | 2020s | OFDMA+mMIMO | 20Gbps | 5G NR |

> **真题高频点**：TD-SCDMA 是大唐提出的 3G 标准，与"大唐杯"直接相关。

### 通信方式分类 [真题-第十届A组]

| 方式 | 定义 | 举例 |
|------|------|------|
| 单工 | 只能单向传输 | 广播、电视 |
| **半双工** | 可双向但不能同时 | **对讲机** |
| 全双工 | 可同时双向 | 手机通话 |

> **真题原题**：按照信息传递的方向和时间关系，普通对讲机采用的通信方式为？→ **B. 半双工通信**
> **真题判断**：对讲机既可以工作在单工方式也可以工作在双工方式 → **正确** [第十届A组]

## 二、5G三大应用场景（重中之重）

### eMBB — 增强移动宽带

| 指标 | 要求 |
|------|------|
| 峰值速率 | 下行20Gbps，上行10Gbps |
| 用户体验速率 | 下行100Mbps |
| 移动性 | 500km/h |

- 典型应用：4K/8K视频、VR/AR、云游戏

### uRLLC — 超可靠低时延通信

| 指标 | 要求 |
|------|------|
| 端到端时延 | 1ms |
| 可靠性 | 99.999%（5个9） |

- 典型应用：自动驾驶、远程医疗、工业控制

> **真题原题**：对于5G应用的三大场景而言，车联网属于哪种场景？→ **D. uRLLC** [第十届A组]

### mMTC — 大规模机器类通信

| 指标 | 要求 |
|------|------|
| 连接密度 | 100万设备/km² |
| 电池寿命 | 10-15年 |

- 典型应用：智能抄表、传感器网络

### 三大场景对比（必背）

| 维度 | eMBB | uRLLC | mMTC |
|------|------|-------|------|
| 核心指标 | 高速率 | 低时延+高可靠 | 海量连接 |
| 典型应用 | VR/8K | 自动驾驶 | 智能抄表 |

> **真题判断陷阱**：eMBB 场景下端到端时延的维度不包括 ftp 服务器延时 [第九届B组]

## 四、5G无线关键技术 [真题-第九届A组]

| 技术 | 说明 | 是否属于5G无线技术 |
|------|------|-------------------|
| 大规模天线阵列 | Massive MIMO | ✓ |
| 超密集组网(UDN) | 提升容量 | ✓ |
| 新型多址 | NOMA等 | ✓ |
| 全频谱接入 | 毫米波等 | ✓ |
| **网络功能虚拟化(NFV)** | 核心网技术 | **✗ 不属于无线技术** |

> **真题原题**：NR系统中，下列哪项不属于5G无线技术？→ **D. 网络功能虚拟化** [第九届A组]
> **注意区分**：NFV是核心网/架构层面技术，不是无线接入技术。

## 五、自由空间传播 [真题-第九届B组]

> **真题原题**：无线信号在自由空间的衰落情况：传播距离每增大一倍，信号衰落增加？→ **C. 6dB**

公式：自由空间路径损耗 L = 32.44 + 20lg(d) + 20lg(f)，距离翻倍 → 20lg(2) ≈ 6dB
