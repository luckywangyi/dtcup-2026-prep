## 二、切换流程 [真题考点]

### Xn切换第一条信令

> **真题原题**：基于基站间Xn链路切换，源基站向目标基站发送第一条信令为？→ **D. Handover Request** [第九届A组]

### 切换流程步骤

```
1. UE上报测量报告(A3事件)
2. 源gNB → 目标gNB: Handover Request
3. 目标gNB → 源gNB: Handover Request Acknowledge
4. 源gNB → UE: RRCReconfiguration(切换命令)
5. UE同步目标小区 + 随机接入
6. UE → 目标gNB: RRCReconfigurationComplete
7. 目标gNB → AMF: Path Switch Request
8. 源gNB释放UE资源
```

### 过覆盖与下倾角 [真题判断-第十届A组]

> **真题判断**：对于过覆盖优化，通常采用压下倾角，此举可以抑制过覆盖，同样也可以缓解灯下黑 → **正确** [第十届A组]
