# 07 5G基站产品及解决方案

> 对应学唐平台课程：03 5G基站产品及解决方案-移广

## 核心知识点

### 一、大唐5G基站产品型号

#### BBU

| 产品 | 型号 | 关键参数 |
|------|------|----------|
| BBU | **EMB6216** / DTTP-6216 | 最大**6**块HBPOF基带板 |

#### BBU板卡

| 板卡 | 功能 |
|------|------|
| **HBPOD** | 基带板（旧型号） |
| **HBPOF** | 基带板（新型号） |
| **HSCTD** | 主控传输板 |
| MPU | 主控板 |

- HSCTD板卡上的接口：**GPS接口** 和 **LMT接口**（不含IR接口、时钟级联接口）
- 板卡不可用原因：时钟问题、板卡规划错误、插入板卡类型不对、处理器加载失败（全选）

#### AAU型号选择

| 场景 | AAU型号 | MIMO | 说明 |
|------|---------|------|------|
| 城区宏站 | 64T64R | Massive MIMO | 设备名称末位识别NR频段 |
| 郊区 | 32T32R | Massive MIMO | |
| 室内 | 4T4R/8T8R | 普通MIMO | |

- AAU不正常接入的因素：DCPD供电开关未闭合、光模块插反、HBPOD板卡未初始化
- **修改小区物理ID列表**不影响AAU正常接入

### 二、供电系统

| 设备 | 供电方式 |
|------|----------|
| BBU | **直流-48V**（注意负号） |
| AAU | **直流-48V** |

- BBU**不是**交流110V或交流220V
- BBU接地线：**16平方**黄绿接地线

### 三、光纤与光模块

| 光纤 | 波长 | 使用场景 |
|------|------|----------|
| 单模1310nm | 黑色 | N78/N79频段，长距离 |
| 多模850nm | — | 短距离 |

- 单模集束光纤在BBU与AAU路由长度超过 **100m** 时使用
- 光模块一般选 **25G-1310**
- BBU侧发送光功率最小值不能低于 **-10dBm**

### 四、GPS天线安装

- 竖直向上视角应大于 **120°**
- 在塔上安装应使其位于铁塔**最面向南方**一角，保持与铁塔 **2米** 间距
- 周围对天线的遮挡不超 **30°**
- ~~两个GPS天线间距大于1米~~ ✗
- 国内运营商5G网络时钟源优先级最高的**不是北斗**，是**GPS**
- 大唐5G基站时钟同步时至少需要锁定 **4颗** 卫星

### 五、BBU安装规范

- 竖装前方需留有 **800mm** 维护通道
- 支持并柜或并架安装
- 双排或多排机柜安装时前后间隔 ≥ **800mm**，同风向安装
- ~~机柜后部不支持靠墙安装与墙距离≤800mm~~ ✗

### 六、AAU安装

- 安装在铁塔/抱杆
- 方位角精度 ±5°，下倾角精度 ±0.5°
- IP65防水，工作温度-40°C~+55°C

### 七、传输配置

#### 必须添加的三条路由

| 路由 | 用途 |
|------|------|
| 基站 → **AMF** | 控制面信令 |
| 基站 → **UPF** | 用户面数据 |
| 基站 → **OMC** | 网管管理 |

- 基站到业务服务器的路由**不是**必须配置的

#### 本地小区建立前提

传输资源正常 + GPS正常 + 基带资源正常 + 射频资源正常（全选）

### 八、巡检工具

| 工具 | 用途 |
|------|------|
| **LMT** | 本地维护终端 |
| **uem5000** | 网管系统 |
| **NodeBSpider** | 基站巡检工具 |

- ~~OSP~~ 不是大唐巡检工具

### 九、安全施工规范

| 操作 | 是否规范 |
|------|----------|
| 上塔前检查安全绳+防坠器 | ✓ 规范 |
| 佩戴防静电手环插拔板卡 | ✓ 规范 |
| 塔下作业佩戴安全帽 | ✓ 规范 |
| **直接拉MPO光纤** | **✗ 不规范** |

### 十、5G小区标识

- 5G NR小区全局标识(NCGI)组成：MCC + MNC + gNB ID + Cell ID

## 考点速记

| 考点 | 记忆要点 |
|------|---------|
| EMB6216最大板数 | 6块HBPOF |
| HSCTD接口 | GPS + LMT |
| 供电 | 直流-48V（注意负号） |
| 接地线 | 16平方黄绿线 |
| 光纤超过多少米用单模 | 100m |
| 光模块 | 25G-1310 |
| GPS天线视角 | >120° |
| 时钟源优先级 | GPS > 北斗 |
| 卫星锁定数 | 至少4颗 |
| BBU维护通道 | 800mm |
| 必配路由 | AMF + UPF + OMC |
| 巡检工具 | LMT/uem5000/NodeBSpider |
| 不安全操作 | 直接拉MPO光纤 |

## 相关真题

> **单选**：EMB6216最大支持几块HBPOF基带板？→ **6** [第九届B组]

> **多选**：HSCTD板卡上接口？→ **GPS接口和LMT接口** [第十届A组]

> **多选**：板卡不可用原因？→ **全选：时钟/规划错误/类型不对/加载失败** [第九届B组]

> **单选**：不影响AAU正常接入的因素？→ **修改小区物理ID列表** [第九届A组]

> **多选**：5G基站供电类型不正确的是？→ **交流110V/直流48V(无负号)/交流220V** [第十届A组]

> **单选**：BBU接地线几平方？→ **16平方** [第九届A组]

> **单选**：单模光纤超过多少米使用？→ **100m** [第九届A组]

> **多选**：GPS天线安装要求正确的？→ **视角>120°/面向南方2米间距/遮挡不超30°** [第九届B组]

> **判断**：时钟源优先级最高是北斗？→ **错误（是GPS）** [第九届A组]

> **多选**：BBU竖装规范？→ **前方800mm/支持并柜/前后≥800mm同风向** [第九届B组]

> **多选**：商用基站必须添加的路由？→ **AMF/UPF/OMC** [第十届A组]

> **多选**：本地小区建立前提？→ **全选：传输/GPS/基带/射频** [第十届A组]

> **多选**：巡检工具？→ **LMT/uem5000/NodeBSpider** [第九届A组]

> **单选**：不符合安全规范的操作？→ **直接拉MPO光纤** [第十届A组]

> **判断**：NCGI包含MCC/MNC/gNB ID/Cell ID？→ **正确** [第九届B组]
