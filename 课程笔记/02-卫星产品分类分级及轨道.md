# 02 卫星产品分类分级及轨道

> 对应学唐平台课程：卫星产品分类分级及轨道

## 核心知识点

### 一、卫星按轨道高度分类

| 类型 | 轨道高度 | 周期 | 特点 |
|------|---------|------|------|
| **LEO（低轨）** | 160-2,000 km | 90-120分钟 | 时延低、路径损耗小、需组网 |
| **MEO（中轨）** | 2,000-35,786 km | 数小时 | 覆盖中等、导航系统常用 |
| **GEO（高轨/同步）** | ~36,000 km | 24小时 | 相对地面静止、覆盖范围大 |
| **HAPS（高空平台）** | 20-50 km | — | 平流层飞行器，非卫星 |

#### 各类轨道对比

| 对比项 | LEO | MEO | GEO |
|--------|-----|-----|-----|
| 时延 | **最低**（几ms~十几ms） | 中等 | **最高**（~270ms单程） |
| 单星覆盖 | 小 | 中 | **最大**（半球） |
| 组网需求 | 需大量卫星组网 | 中等 | 3颗可覆盖全球 |
| 系统复杂度 | **最高** | 中 | 低 |
| 频率复用 | 高 | 中 | 低 |
| 发射成本 | 单颗低 | 中 | 单颗高 |

- 卫星互联网通信中，**低轨卫星（LEO）**的性能更适配卫星互联网发展
- GEO（静止卫星）的轨道**必然处于赤道平面上**
- **不是**必须使用静止卫星才能实现卫星通信（LEO/MEO也可以）

### 二、卫星通信频段

| 频段 | 频率范围 | 特点 | 典型应用 |
|------|---------|------|----------|
| **L频段** | 1-2 GHz | 穿透力强、抗雨衰、带宽低 | 海事、应急、天通一号 |
| **S频段** | 2-4 GHz | 抗雨衰较强、中低速率 | 中低速数据 |
| **C频段** | 4-8 GHz | 抗雨衰强、技术成熟 | 固定卫星通信主流 |
| **Ku频段** | 12-18 GHz | 高带宽、商用成熟 | 卫星电视/宽带 |
| **Ka频段** | 26.5-40 GHz | 超大带宽、雨衰极强 | 高通量卫星 |

- 常用的固定卫星以**C频段**居多，C频段为上行6GHz、下行**4GHz**
- 卫星移动通信业务使用 **L频段**
- 卫星地球站通过赤道上空约36,000km的卫星转发，一跳单程空间距离为72,000-**80,000km**

### 三、中国卫星通信系统

#### 天通一号

| 参数 | 说明 |
|------|------|
| 类型 | **GEO同步轨道**卫星 |
| 用途 | 卫星移动通信 |
| 运营商 | **中国电信** |
| 卫星数量 | 目前由**3颗**通信卫星支撑 |
| 代表终端 | 华为Mate60 Pro卫星通话功能 |

- 华为Mate60 Pro卫星通话采用天通一号系统
- 天通一号是**GEO同步**卫星（不是LEO）
- 由**中国电信**运营（不是中国移动或联通）
- 华为Mate60 Pro卫星通话**不需要**先将信号发给地面站（直接连卫星）

#### 北斗导航系统

- 由MEO/GEO/IGSO三种轨道卫星组成
- GPS接收装置利用 **4颗** 以上卫星信号定位

### 四、卫星电话系统组成

卫星电话系统需要包括：
- 通信卫星组成的链路 ✓
- 地面站设备 ✓
- 用户终端设备 ✓
- **卫星天线设备**不属于系统基本组成（是终端的一部分）

卫星电话手机与普通手机相比，需要额外的**卫星通信天线和硬件**来支持与卫星通信系统进行通信。

### 五、数字卫星通信方式

属于数字卫星通信方式：
- 不加话音插空的时分多址 ✓
- 卫星数字通道方式 ✓
- 数字话音插空的时分多址 ✓

### 六、6G超低轨卫星

- 6G超低轨道卫星距地面高度约 **300** 公里
- 超低轨可进一步降低时延、提高频率复用效率

## 考点速记

| 考点 | 记忆要点 |
|------|---------|
| LEO高度 | 160-2000km |
| GEO高度 | ~36000km |
| GEO轨道位置 | 赤道平面上 |
| 适配互联网 | LEO低轨 |
| C频段 | 固定卫星主流，上行6G/下行4G |
| L频段 | 卫星移动通信 |
| 天通一号类型 | GEO同步轨道 |
| 天通运营商 | 中国电信 |
| 天通卫星数 | 3颗 |
| Mate60通话 | 不经地面站，直连卫星 |
| GPS定位 | 至少4颗卫星 |
| 6G超低轨 | ~300km |

> **判断**：静止卫星轨道必然处于赤道平面上 → **正确** [第十一届研究生]

> **单选**：常用固定卫星C频段下行多少GHz？→ **4GHz** [第十一届A组]

> **单选**：卫星移动通信业务使用哪个频段？→ **L频段** [第十一届A/研究生组]

> **单选**：华为Mate60 Pro卫星通话由谁运营？→ **中国电信** [第十一届B组]

> **单选**：天通一号目前由几颗卫星支撑？→ **3颗** [第十一届A组]

> **判断**：Mate60 Pro卫星通话需要先发给地面站？→ **错误** [第十一届高职]

> **单选**：GPS定位需几颗以上卫星？→ **4颗** [第十一届研究生]
