# 04 通感技术应用及架构介绍

> 对应学唐平台课程：11 通感技术应用及架构介绍

## 核心知识点

### 一、通感一体化（ISAC）概述

ISAC（Integrated Sensing and Communication）是5G-A/6G的重要创新技术，通过复用一套设备、频谱和硬件平台，实现**通信与感知功能的深度融合**。

#### 核心理念

- **频谱共用**：通信与感知共享同一频段
- **硬件共用**：同一组天线、射频前端同时执行通信和感知
- **软件协同**：通信协议与雷达算法融合处理
- "一网多能"：共站址、共设备、共频段、共运维

### 二、技术原理

#### 感知机制

基站发射通信信号 → 信号被目标物体反射 → 基站接收反射回波 → 提取距离/速度/角度信息

与雷达原理类似，利用回波信号实现：

| 感知参数 | 提取方法 |
|---------|---------|
| **距离** | 回波时延 |
| **速度** | 多普勒频移 |
| **角度** | 多天线到达角（AoA） |
| **位置** | 距离+角度联合计算 |

#### 通感信号融合方式

| 方式 | 说明 |
|------|------|
| **时分复用** | 通信和感知在不同时隙交替 |
| **频分复用** | 通信和感知使用不同子载波 |
| **全双工** | 同时发射通信信号和接收感知回波 |
| **波形共享** | 同一OFDM波形同时承载通信数据和感知功能 |

### 三、系统架构

#### 三层架构

```
感知数据融合分析（云端/边缘计算平台）
              ↑
    通感一体基站（gNB + 感知引擎）
              ↑↓
     终端/目标物体（UE/车辆/无人机等）
```

#### 大唐相关产品

| 产品 | 说明 |
|------|------|
| **HMCPB（通感板）** | 专用通感感知板卡 |
| **HMCPC（融合板）** | 通信与感知融合板卡 |

### 四、典型应用场景

#### 低空经济

- **无人机物流**：航线感知与路径规划
- **低空安防**：飞行器实时监测与非法飞行物告警
- **文旅交通**：低空飞行器管理

#### 车路协同

- 车辆位置和速度感知
- 行人检测与碰撞预警
- 交叉路口安全辅助

#### 海面/航道管理

- 船只探测与航行状态感知
- 非法入侵警告
- 港口安全监控

#### 地空一体

- 机场跑道入侵检测
- 飞鸟检测与驱离
- 高精度定位

#### 其他

- 桥梁微形变监测
- 室内入侵检测
- 行人密集区域检测
- 智慧工厂环境感知

### 五、关键性能指标

| 指标类型 | 具体指标 |
|---------|---------|
| **感知性能** | 位置精度（厘米级）、分辨率、速度范围/精度、虚警率、漏检率 |
| **通感协同** | 通信与感知的资源分配效率、互干扰抑制能力 |

### 六、与5G-A的关系

- 通感一体化是 **5G-A（Release 18-20）** 的核心特征之一
- 5G-A三大新能力：**上行增强、通感一体、AI原生**
- 3GPP在R19中进一步推进通感标准化
- 通感一体化也是6G的关键技术方向

### 七、与传统雷达的区别

| 对比项 | 传统雷达 | 通感一体 |
|--------|---------|---------|
| 专用设备 | 需独立雷达系统 | 复用通信基站 |
| 频谱 | 专用雷达频段 | 共享通信频段 |
| 成本 | 高 | 低（共用基础设施） |
| 覆盖 | 局部部署 | 随通信网络广覆盖 |
| 精度 | 高（专用优化） | 达到厘米级（持续提升） |
