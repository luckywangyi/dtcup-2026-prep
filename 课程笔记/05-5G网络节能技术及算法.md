# 05 5G网络节能技术及算法

> 对应学唐平台课程：08 DTM.PX31.023.017-5G网络节能技术及算法

## 核心知识点

### 一、5G基站功耗分布

- AAU功耗占整机 **90%**
- AAU是节能的主要对象

### 二、四级节能技术体系

| 技术 | 原理 | 节能效果 | 恢复时间 |
|------|------|----------|----------|
| **符号关断** | 无数据发送时关断部分OFDM符号的功放 | 20-30% | 毫秒级 |
| **通道关断** | 动态关闭部分天线通道（如64T→32T） | 30-40% | 秒级 |
| **载波关断** | 低负载时关闭部分载波频点 | 40-50% | 秒级 |
| **深度休眠** | 关闭功放、收发信机等模拟器件 | 65%+ | 5分钟内 |

#### 更深层次休眠

| 层级 | 关闭范围 | 节能效果 |
|------|---------|----------|
| 浅层休眠 | 功放、收发信机 | 30%+ |
| 深度休眠 | +中频业务处理器件+BBL数据处理模块 | 65%+ |
| 极致休眠 | 仅保留电源/定时/控制电路 | 功耗<10W |

### 三、通道关断的影响

通道关断会导致：
- 小区覆盖范围减小 ✓
- 误块率上升 ✓
- 用户业务吞吐量降低 ✓
- **不会**导致符号关断节能终止生效

### 四、符号节能进入/退出条件

| 条件 | 门限 |
|------|------|
| **进入节能** | RRC连接数低于**低门限** + PRB利用率低于低门限 |
| **退出节能** | RRC连接数高于**高门限** 或 手动关闭 或 超出设定时间段 |

退出符号节能条件（全选ABC）：
- A. 时间段超出设定时间段 ✓
- B. 手动关闭节能开关 ✓
- C. RRC连接数**高于**高门限 ✓
- ~~D. PRB利用率高于低门限~~ ✗（应为高于**高**门限）

### 五、AI智能节能算法

#### 基于负荷预测的AI节能

- 通过小波神经网络模型预测基站流量
- 根据预测结果提前执行休眠/唤醒策略
- 效果：节能真实生效时间提升80%+，能耗降低约10%

#### 基于强化学习的节能

- 使用Sarsa强化学习算法
- 状态：用户数、基站负载
- 动作：休眠/唤醒
- 奖励：节能量-服务质量损失

#### 基于业务导航的协同节能

- 通过调整用户在多频多制式网络间的分布
- 将用户集中导向少量载波，关断空闲载波
- 实现多频多制式协同节能

### 六、核心网协同节能

- 策略控制功能（PCF）基于全局策略生成节能指令
- 综合考虑：实时业务分布、用户QoS等级
- 通过大数据预测和业务触发实现节能与体验平衡

### 七、SON自组织网络

| 功能 | 说明 |
|------|------|
| 自配置 | 新站自动参数配置 |
| 自优化 | 自动调整覆盖/容量/切换参数 |
| 自愈 | 故障后自动补偿 |
