# 09 工程概论基础-信科赛专项培训

> 对应学唐平台课程：工程概论基础-信科赛专项培训

## 核心知识点

### 一、IPD集成产品开发

#### IPMT（集成产品管理团队）

- IPMT负责产品**投资决策**
- 将产品策划和立项看作**投资行为**

IPMT决策的关键内容：
- 产品最终综合竞争力 ✓
- 投资回报率 ✓
- 商业模式 ✓
- ~~技术因素~~ ✗（技术是手段不是决策核心）

- IPMT流程第二阶段的输出成果：**新产品立项批准报告**

### 二、产品成本

#### 产品全成本概念

- 产生开发成本的环节：**产品设计**

#### 财务会计 vs 管理会计

| | 财务会计 | 管理会计 |
|---|---------|---------|
| 定义 | 企业为生产一定种类数量的产品所支出的生产费用之和 | 产品形成中及交付后仍由企业支付的生产服务费用之和 |
| 报表使用者 | 企业**外部**（投资者、监管） | 企业**内部**管理者 |

#### 产品双重属性

- 产品具有 **技术属性** 和 **经济属性**
- 不能只追求技术先进性，要兼顾经济性（性价比）

### 三、技术创新

| 创新方式 | 说明 | 难度 |
|----------|------|------|
| **原始创新** | 从零开始的创新 | **最高** |
| 集成创新 | 整合现有技术 | 中 |
| 改进创新 | 改进现有产品 | 较低 |

最有技术难度的创新方式：**原始创新**

### 四、专利策略

发现核心技术已被他人申请专利，可采取：
- 专利避让 ✓
- 交叉许可 ✓
- 交纳专利使用费 ✓
- ~~不影响继续申请专利~~ ✗

### 五、通信系统概念

#### 电话系统呼叫消息

| 消息 | 含义 |
|------|------|
| IAM | 初始地址消息 |
| ACM | 地址全消息 |
| CLF | 前向拆线信号 |
| **ANC** | **被叫应答消息**（不是主叫） |

#### 蜂窝移动通信应用场景

属于蜂窝移动通信应用场景：路云通信、车路通信、车网通信
- ~~人云通信~~ 不属于

### 六、通信工程项目管理

#### 项目生命周期

需求分析 → 方案设计 → 工程施工 → 验收交付 → 运维优化

#### 项目管理要素

- 范围管理：明确项目边界
- 时间管理：进度计划与控制
- 成本管理：预算与费用控制
- 质量管理：标准与验收
- 风险管理：识别、评估与应对

#### 通信工程验收

- 初验（竣工验收）
- 终验（试运行后验收）
- 验收内容：设备安装、线缆施工、系统调测、文档交付

## 考点速记

| 考点 | 记忆要点 |
|------|---------|
| IPMT决策 | 竞争力/ROI/商业模式（不含技术） |
| IPMT第二阶段输出 | 新产品立项批准报告 |
| 开发成本 | 产生于产品设计环节 |
| 管理会计使用者 | 企业内部 |
| 产品属性 | 技术+经济双重属性 |
| 最难创新 | 原始创新 |
| 专利应对 | 避让/交叉许可/交费 |
| ANC | 被叫应答（不是主叫） |
| 非蜂窝场景 | 人云通信 |

## 相关真题

> **多选**：IPMT决策关键内容？→ **综合竞争力/投资回报率/商业模式** [第十届A组]

> **单选**：IPMT第二阶段输出？→ **新产品立项批准报告** [第十届A组]

> **单选**：产品全成本中开发成本产生环节？→ **产品设计** [第十届A组]

> **多选**：关于产品成本描述正确的？→ **管理会计定义+财务会计定义** [第十届A组]

> **判断**：管理会计报表使用者是企业外部？→ **错误（是内部）** [第十届A组]

> **单选**：产品具有经济属性和？→ **技术属性** [第十届A组]

> **判断**：产品设计不能只追求技术先进性？→ **正确** [第十届A组]

> **单选**：最有技术难度的创新方式？→ **原始创新** [第十届A组]

> **单选**：核心技术被专利保护，不可采取的措施？→ **不影响继续申请** [第十届A组]

> **单选**：电话系统呼叫消息不正确的是？→ **ANC是主叫应答（应为被叫）** [第十届A组]

> **单选**：不属于蜂窝移动通信应用场景的？→ **人云通信** [第十届A组]
