# 01 卫星网络架构及接口介绍

> 对应学唐平台课程：卫星网络架构及接口介绍

## 核心知识点

### 一、NTN（非地面网络）概述

NTN（Non-Terrestrial Network）是3GPP将卫星通信纳入5G标准体系的技术框架，实现天地一体化网络覆盖。

#### NTN网络组成

| 组件 | 说明 |
|------|------|
| **NTN平台** | 卫星（LEO/MEO/GEO）或高空平台（HAPS） |
| **NTN网关** | 地面站，连接卫星与地面核心网 |
| **用户终端（UE）** | 支持NTN的手机或IoT设备 |
| **地面核心网** | 5G核心网（AMF/UPF等），与地面网络共用 |

### 二、关键链路定义

| 链路 | 说明 |
|------|------|
| **Service Link（服务链路）** | 卫星 ↔ 用户终端（UE）之间的链路 |
| **Feeder Link（馈电链路）** | NTN网关 ↔ 卫星之间的上行/下行链路 |
| **ISL（星间链路）** | 卫星与卫星之间的通信链路 |

```
            ┌──────────┐
            │  卫星/HAPS │
            └──┬───┬───┘
  Feeder Link │   │ Service Link
               │   │
      ┌────────┘   └────────┐
      │                     │
┌─────┴─────┐         ┌────┴────┐
│ NTN网关    │         │   UE    │
│ (地面站)   │         │(终端)   │
└─────┬─────┘         └─────────┘
      │
┌─────┴─────┐
│ 5G核心网   │
│ (AMF/UPF) │
└───────────┘
```

### 三、两种转发架构

#### 透明转发（Bent-Pipe/透明模式）

- 卫星仅进行**频率转换、滤波和放大**，无板载处理
- 相当于模拟射频中继器
- 5G Uu接口在**地面基站（gNB）**处终止，不在卫星处终止
- gNB功能完全在地面实现
- **3GPP R17版本主要支持方案**

#### 再生转发（非透明模式）

- 卫星具备**解调/解码、交换/路由、编码/调制**等板载处理能力
- 卫星部分或全部搭载gNB功能
- 支持星间链路（ISL），实现空间组网
- **R18+版本**中规划推进

| 对比项 | 透明转发 | 再生转发 |
|--------|---------|---------|
| 卫星处理能力 | 仅RF层面 | 具备基带处理 |
| gNB位置 | 地面 | 部分/全部在星上 |
| 复杂度 | 低 | 高 |
| 灵活性 | 低 | 高 |
| 标准阶段 | R17 | R18+ |
| ISL支持 | 不支持 | 支持 |

### 四、3GPP NTN标准化进程

| 版本 | 时间 | 内容 |
|------|------|------|
| **R15-R16** | 2018-2020 | 研究阶段：定义部署场景、信道模型 |
| **R17** | 2022 | 第一个NTN标准：透明转发架构下终端直连 |
| **R18** | 2024 | 增强：移动性、服务连续性、覆盖增强、UPF上星 |
| **R19** | 2025+ | 星上处理、跳波束等技术研究 |

- NTN标准从 **R15** 开始启动研究（3GPP未来将向6G演进）

### 五、NTN应用场景

| 场景 | 说明 |
|------|------|
| **连续服务** | 航空、海事、偏远地区覆盖延伸 |
| **泛在服务** | 全球无缝覆盖，包括海洋/沙漠/极地 |
| **扩展服务** | 物联网广域覆盖、应急通信 |
| **定位增强** | 结合卫星提供高精度定位 |

### 六、与地面网络的融合

- NTN与地面5G网络共享核心网基础设施
- 可实现地面网络与卫星网络之间的无缝切换
- 终端需要支持NTN功能（如时延预补偿、多普勒补偿）

## 考点速记

| 考点 | 记忆要点 |
|------|---------|
| NTN组成 | 卫星+网关+UE+核心网 |
| Service Link | 卫星↔UE |
| Feeder Link | 网关↔卫星 |
| 透明转发 | 仅RF中继，gNB在地面，R17 |
| 再生转发 | 星上基带处理，gNB在星上，R18+ |
| NTN标准起始 | R15开始研究 |
| 第一个NTN标准 | R17 |
| ISL | 星间链路，再生转发支持 |
