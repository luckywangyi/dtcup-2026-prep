## 核心知识点

### 一、日常运维流程（概括）

**监控 → 告警受理 → 分级派单 → 远程/现场处理 → 变更与复盘 → 报告与知识库更新**。

与 **网管、LMT、工单系统** 联动；重大操作遵守 **窗口期、双人复核、回退预案**。

### 二、运维工具（07 与真题）

| 工具 | 用途 |
|------|------|
| **LMT** | 本地维护终端 |
| **uem5000** | 网管系统 |
| **NodeBSpider** | 基站巡检工具 |

**说明**：07 课件指出 **OSP 不是大唐基站巡检工具**；若试卷选项或外部资料出现 OSP，**以学唐课件与官方解析为准**。

### 三、故障分类分级（工程常用模型）

常见 **三级**（企业可自定义名称）：

| 级别 | 典型特征 | 响应要求（理解） |
|------|----------|------------------|
| **一级** | 大面积业务中断、核心设备瘫痪 | 最高优先级，立即升级 |
| **二级** | 局部区域质差、重要集客受影响 | 限时恢复 |
| **三级** | 单站或次要问题 | 计划内处理 |

### 四、设备监控与告警处理

- **监控对象**：小区退服、光口误码、时钟失锁、温度、电源、CPU/内存、KPI 劣化。  
- **处理思路**：告警关联压缩 → 根因分类（传输、电源、天馈、参数、硬件）→ 工具验证 → 变更闭环。

### 五、覆盖优化

| 问题 | 表现与思路 |
|------|------------|
| **弱覆盖** | RSRP 低；加站、功率、天馈、下倾/方位、波束参数 |
| **越区覆盖** | 远端仍强信号；压下倾、降功率、天线指向、增加阻挡 |
| **重叠覆盖** | 多小区强度接近；功率/下倾/邻区协同 |
| **导频污染** | 无主导频，SINR 差；控制邻区数目与相对强度 |

**真题**：覆盖问题包括 **弱覆盖、导频污染、越区覆盖** 等（高职组多选）。

**路测弱覆盖**：原因可能是功率、天馈、参数、阻挡等；**不一定**与邻区漏配无关（判断题：说“肯定无关”常为 **错**）。

### 六、干扰优化

- **邻区干扰**：PCI/mod3、重叠、越区导致 **SS-SINR** 恶化。  
- **远端干扰、杂散、互调**：按频段与设备排查。  
- **mod3/mod4 与 SS-SINR**（判断题）：不同试卷结论需按 **原题解析**。

### 七、切换优化与 A3 事件

- **A3**：邻区质量优于服务小区一定偏置，常用于同频切换测量报告。  
- **抑制切换**（第八届真题）：**增大 A3 偏移**、**增大上报间隔** 等可 **抑制** 过早/过频切换；**减小触发时间** 可能促发更灵敏上报（选项需细算）。

**异频测量**：**A2** 等事件开启测量（研究生组真题）。

### 八、关键无线指标

| 指标 | 含义（备考） |
|------|--------------|
| **RSRP** | 参考信号接收功率，覆盖强度 |
| **RSRQ** | 参考信号接收质量，含负载因素 |
| **SINR** | 信干噪比，吞吐与 MCS 强相关 |

空闲态 **RSRP** 测量对象（第九届真题）：与 **SSB** 相关测量配置对应（选项以试卷为准）。

**RRC_CONNECTED** 下可测量 **SSB 与 CSI-RS** 的 RSRP/RSRQ/SINR（本科 B 组判断）。

### 九、链路预算与传播模型

- **链路预算**：发射功率 + 天线增益 − 线缆损耗 − 传播损耗 − 余量 → 估算 **最大允许路径损耗** → **小区半径**（与 **传播模型** 联立）。  
- **真题**：链路预算需与 **传播模型** 等结合得到小区半径。

常用模型概念：**Cost231-Hata、Okumura-Hata、3GPP UMa/RMa** 等（细节以教材与题目给定公式为准）。

### 十、AI/ML 在网络优化中的应用（08 扩展）

**机器学习流程**（多届真题）：**数据收集 → 清洗 → 特征工程 → 建模**。

**AI 赋能 5G**（真题）：**资源调度、网络优化**；**不是**用 NLP/语音识别代替 5G 本身。

**在网络优化中的典型方向**：

- **智能节能**：负荷预测 + 载波/通道关断策略。  
- **智能调参**：邻区、切换、功率参数推荐；**5G 智能网络优化调参** 题目涉及 **训练/验证/测试集**（本科 A 组）。  
- **异常检测**：KPI 异常、隐性故障、光模块劣化趋势。  
- **根因定位**：告警与 MR 多维关联分类。

**集成学习**（第十届 / 第十一届）：**Boosting、Bagging、Stacking、Voting** 均属集成学习；**随机森林** 属 **Bagging**。
