## 核心知识点

### 一、NTN（Non-Terrestrial Network，非地面网络）

将 **卫星、高空平台（HAPS）** 等接入纳入与地面蜂窝 **统一设计与互操作** 的框架；**3GPP** 自 **R15** 启动 NTN 研究，**R17** 首个完整 NTN 规范基线，**R18+** 强化星上处理与架构演进。

### 二、NTN 架构：透明转发 vs 再生转发

| 对比项 | **透明转发（Bent-Pipe）** | **再生转发（星上处理）** |
|--------|---------------------------|---------------------------|
| 卫星能力 | **射频滤波、变频、放大** | + **调制解调、交换路由** 等基带处理 |
| gNB 位置 | **地面**（NTN Gateway / 信关站侧） | **星上**（部分或全部） |
| ISL | 一般 **无** | **可支持** 星间链路组网 |
| 复杂度 | 低 | 高 |
| 标准阶段 | **R17** 主力 | **R18+** 推进 |

**记忆**：透明转发像 **“射频镜子”**；再生转发像 **“太空基站”**。

### 三、关键链路

| 链路 | 英文 | 两端 |
|------|------|------|
| **服务链路** | Service Link | 卫星 ↔ **UE** |
| **馈电链路** | Feeder Link | 卫星 ↔ **信关站/网关** |
| **星间链路** | ISL | 卫星 ↔ 卫星 |

无 ISL 时，**馈电链路** 为地面与空间段 **必经** 回传。

### 四、轨道与频段（精华）

**轨道**：

- **LEO**：约 **160–2000 km**，时延低，需星座组网；卫星相对地面高速运动（约 **7.5–7.8 km/s**），切换与波束管理复杂。  
- **MEO**：中轨，导航等常见。  
- **GEO**：约 **36000 km**，相对地面静止，单星覆盖大；**往返时延约 250–400 ms** 量级，不利实时交互。  
- **卫星互联网**：**LEO** 性能更适配大容量低时延发展（真题）。

**频段（02）**：

- **L 频段**：卫星 **移动通信**、抗雨衰好（**天通**、海事应急）。  
- **C 频段**：固定卫星业务主流；上行 **6 GHz**、下行 **4 GHz**（真题）。  
- **Ku/Ka**：高通量宽带；Ka 雨衰敏感。

### 五、空口挑战与补偿（03）

| 挑战 | 补偿思路 |
|------|----------|
| **大时延** | **定时提前（TA）** 与 **星历+GNSS** 预补偿；HARQ **进程扩展或反馈禁用** |
| **大多普勒** | **大子载波间距**；发射端 **频率预补偿**；联合时频同步 |
| **大路径损耗** | 高增益终端、功率控制、波束赋形 |
| **波束切换** | 基于 **星历** 的可预测切换 |

### 六、部署演进四阶段（01）

1. **透明转发**：gNB 全在地面。  
2. **DU 上星**：CU/DU 分离，DU 在星上。  
3. **gNB 上星**：完整基站上星 + NG over SRI。  
4. **核心网上星**（愿景）：核心网功能天基化。

### 七、星地融合网络：五大融合 → 七大融合

**早期五大融合**（01）：**终端、体制、系统、网络、业务**。

**面向 6G 的七大融合**：**体制、网络、管理、频谱、业务、平台、终端**。

**融合三阶段**：**互联互通**（业务借鉴）→ **混合接入**（资源共享）→ **深度融合**（一致体验）。

### 八、关键接口体系（了解）

卫星专网与高轨/低轨体系课件列出 **S-NR-Uu、S-N2（S-NG-C）、S-N3（S-NG-U）、S-ISL** 等，分别对应 **空口、基站—AMF、基站—UPF、星间路由** 等（细节以 `01` 笔记表格为准）。

### 九、天通一号与产业案例（02）

- **GEO 同步**卫星移动通信系统；**中国电信**运营；**3 颗** 卫星支撑（题库口径）。  
- **Mate 60 Pro** 等终端 **直连卫星** 通话，**无需**先经地面站再转卫星（判断题：**错误** 项为“必须先发地面站”）。  
- **卫星电话系统组成**：**通信卫星链路、地面站、用户终端**；**“卫星天线设备”** 曾作为 **不属于系统基本组成** 的干扰项（天线属终端能力的一部分，表述以试卷为准）。

### 十、3GPP 标准化与创新应用

- **标准化**：R15 启动 → R17 NTN 基线 → R18+ 星上 gNB/ISL。  
- **创新应用**：应急通信、远洋/航空、物联网、偏远宽带、**星地协同车联网**、行业专网等。

## 考点速记

| 考点 | 记忆要点 |
|------|----------|
| NTN 启动 | **R15** 研究；**R17** 首个完整规范节点（题库） |
| 透明 vs 再生 | **地面 gNB** vs **星上 gNB**；**R17** vs **R18+** |
| 三大链路 | Service / Feeder / **ISL** |
| LEO 速度 | 约 **7.8 km/s** |
| GEO 时延 | **250–400 ms** 量级 |
| 卫星互联网轨道 | **LEO** 更适配 |
| 天通 | **GEO**、**电信**、**3 星** |
| 移动卫星频段 | **L** |
| C 频段下行 | **4 GHz** |
| 七大融合 | 体制/网络/管理/频谱/业务/平台/终端 |
| 6G 超低轨高度 | 约 **300 km**（真题） |
