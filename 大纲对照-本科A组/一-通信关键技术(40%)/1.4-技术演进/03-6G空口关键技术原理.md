# 6G空口关键技术原理

> 大纲分类：一、通信关键技术 > 四、技术演进 > 6G空口关键技术原理  
> 考核要求：熟悉

---

## 核心知识点

### 一、太赫兹通信（THz, 0.1–10 THz 量级）

**原理与动机**：更高载波频率意味着**潜在更宽连续带宽**，可支撑 **Tb/s 级峰值** 愿景；同时波长更短，利于**超大规模阵列**集成。

**挑战（熟悉级）**：  
- 传播损耗大、穿透弱，覆盖以**短距热点、室内、回传**等场景为主。  
- 射频器件效率、本振相位噪声、信道测量与建模难度大。  
- 与 **光学/红外** 等共同构成“全频谱”讨论。

**备考定位**：6G **候选频谱方向**，常与 **RIS、ELAA** 联合出题。

### 二、智能超表面（RIS / IRS）

**RIS（Reconfigurable Intelligent Surface）** / **IRS（Intelligent Reflecting Surface）**：由大量可编程无源或有源单元组成的二维表面，通过调控反射/透射系数，**重构无线信道**（波束赋形、干扰抑制、覆盖补盲）。

**特点**：相对传统有源中继，部分架构可**降低功耗与成本**；可与 **ISAC** 结合实现环境感知辅助通信。

**真题高频**：**超材料无源阵列、低功耗调控电磁波** → 指向 **RIS**。

### 三、语义通信（Semantic Communication）

传统通信追求 **比特无误传输**；**语义通信**强调在收发端共享知识库的前提下，只传输**与任务相关的语义信息**，在**语义层面**恢复含义，而非逐比特复制。

**价值**：在 AI 时代降低 **无效比特传输**、提升 **任务完成率** 与能效（尚处研究与标准化早期）。

### 四、新型多址接入：NOMA / RSMA

- **NOMA（Non-Orthogonal Multiple Access）**：在功率域或码域**非正交**复用多用户，接收端用 **SIC** 等算法分离；提升连接数与频谱效率，代价是接收复杂度与干扰管理。  
- **RSMA（Rate-Splitting Multiple Access）**：通过**速率分割+公共/私有流**统一处理多用户干扰，可视为更灵活的多址与干扰管理框架。

**定位**：6G **候选多址** 与 **mMTC/海量接入** 场景的重要研究方向。

### 五、全双工通信（In-band Full Duplex）

在同一频段**同时收发**，理论上可**倍增频谱效率**。

**最大难点**：**自干扰抑制/消除**（天线域、模拟域、数字域联合抑制）。  
（大唐杯真题选项多表述为 **“干扰抑制”**。）

### 六、超大规模 MIMO（ELAA, Extremely Large Aperture Array）

相对 5G **Massive MIMO**，ELAA 进一步增大**阵列孔径**与天线规模，追求更高空间分辨率、更强波束管理与覆盖；常与 **毫米波/太赫兹**、**近场效应** 研究结合。

## 考点速记

| 考点 | 记忆要点 |
|------|---------|
| THz | 极高频段、超大带宽潜力、覆盖挑战 |
| RIS/IRS | 可重构环境、低功耗补盲、超材料阵列 |
| 语义通信 | 传“含义”非全部比特；需知识库 |
| NOMA | 非正交、SIC、频谱效率↑ |
| RSMA | 速率分割、干扰管理更灵活 |
| 全双工 | 难点 = **自干扰消除** |
| ELAA | 超大规模天线/孔径，6G MIMO 演进 |
