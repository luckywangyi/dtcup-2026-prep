## 核心知识点

### 一、Massive MIMO 基本原理

**大规模天线阵列**：在基站侧部署远多于传统系统的天线单元（典型数十至上百根），在同一时频资源上通过**空分复用（SDMA）**同时服务多个空间上可区分的用户。核心思想是利用**信道空间维度的独立性**：不同用户与基站之间的空间信道向量近似正交或低相关，从而在同一 RB 上并行传输多路数据流，显著提升**频谱效率**与**小区容量**。

**与传统 MIMO 的区别（速览）**：

| 对比项 | 传统 MIMO（如 4T4R） | Massive MIMO |
|--------|----------------------|--------------|
| 天线规模 | 少量天线 | 数十～上百通道（如 32T、64T） |
| 波束形态 | 宽波束为主 | **窄波束、可扫描**，能量更集中 |
| 复用方式 | 有限 MU-MIMO | 更强 **MU-MIMO / 空分**能力 |
| 覆盖与干扰 | 依赖宏分集等 | **波束赋形**显著改善边缘与邻区干扰 |
| 工程复杂度 | 相对较低 | 需校准、权值管理、波束流程 |

### 二、波束赋形：模拟 / 数字 / 混合

- **模拟波束赋形（Analog BF）**：在射频端用移相器形成波束，硬件简单、功耗较低，但**同时多波束/多流能力受限**，灵活性差。
- **数字波束赋形（Digital BF）**：每根天线独立基带权重（**天线权值**），可形成任意指向与多用户并行波束，**灵活度最高**，但基带与收发链路数量大、成本高。
- **混合波束赋形（Hybrid BF）**：模拟部分做粗波束/宽角覆盖，数字部分做精细权值与多用户调度，在 **5G 毫米波与大规模阵列**中常见，平衡性能与成本。

**天线权值**：描述各天线单元幅度与相位的系数集合；网络根据**信道估计（如 SRS）**、覆盖场景与优化目标计算权值，使能量对准目标 UE 或抑制干扰。工程上常强调**权值与场景匹配**（如高楼、道路、郊区模型）。

### 三、典型配置：64T64R / 32T32R 与场景（工程侧）

结合大唐产品笔记的典型选型：

| 场景 | 典型 AAU | MIMO 类型 | 说明 |
|------|----------|-----------|------|
| 城区宏站 | **64T64R** | Massive MIMO | 容量与垂直/水平维波束能力要求高 |
| 郊区 | **32T32R** | Massive MIMO | 相对降维，仍保留波束增益 |
| 室内 | 4T4R / 8T8R | 普通 MIMO | 覆盖距离短、安装条件不同 |

设备名称末位常可识别 NR 频段；**AAU 一体化**有利于减少馈线损耗、提高**天线校准**精度，从而改善波束赋形效果。

### 四、波束管理与 SSB 波束扫描

- **SSB（Synchronization Signal Block）**：承载 PSS/SSS/PBCH，用于**小区搜索与初始接入**。
- **SSB 波束扫描**：基站用**不同指向的波束**分时发送 SSB，UE 测量各波束 RSRP/RSRQ，上报**波束索引**；网络据此选择服务波束。这与 Massive MIMO **窄波束覆盖**强相关。
- **波束失败恢复（BFR）**：当前服务波束质量恶化时，按规范流程恢复链路。
- 真题常考：**波束扫描定义**（下行依次用不同指向波束发射）、**SSB 可配置波束数量与子帧配比关系**、分级扫描策略等。

### 五、Massive MIMO 在 5G 中的典型价值

- **大容量**：空分复用提升同频同时用户数与吞吐量。
- **深度覆盖与边缘体验**：能量对准 UE，等效提高接收 SNR。
- **天面受限场景**：通过波束“虚拟”增强方向性，缓解站址与天线数物理限制（与“天面不足”类表述在题库中常作为正确选项之一）。
- **MU-MIMO 配对**：基站根据 UE 的 SRS 等估计上行信道相关性，选择**低相关** UE 配对，降低流间干扰。

**注意**：Massive MIMO **不能**单独“解决穿透损耗”本质问题（穿透由材料与频段决定），题库中若选项为“穿透损耗”需结合题干判断是否设错。

## 考点速记

| 考点 | 记忆要点 |
|------|----------|
| 本质 | 大规模阵列 + **空分复用** + 波束赋形 |
| 权值 | 各天线幅度相位；与**覆盖场景**、SRS/CSI 匹配 |
| 波束类型（实现） | 模拟 / 数字 / **混合**（5G 常用折中） |
| 工程配置 | 城区 **64T64R**，郊区 **32T32R**，室内 4T/8T |
| SSB | **波束扫描**接入；波束索引测量与选择 |
| vs 传统 MIMO | 规模更大、窄波束、更强 SDMA/MU-MIMO |
| 部署目的（题库高频） | **大容量、深度覆盖、天面不足**等（穿透损耗慎选） |
| AAU 优势 | 集成度高、**校准**好、波束赋形性能更稳 |
