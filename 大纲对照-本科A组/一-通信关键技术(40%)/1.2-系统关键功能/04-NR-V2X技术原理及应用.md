## 核心知识点

### 一、V2X 概念与分类

**V2X（Vehicle to Everything）** 指车辆与周边实体的信息交互，常见四类：

| 类型 | 含义 | 典型应用 |
|------|------|----------|
| **V2V** | Vehicle to Vehicle | 防碰撞、编队、协作变道 |
| **V2I** | Vehicle to Infrastructure | 信号灯状态、路侧限速、RSU 服务 |
| **V2P** | Vehicle to Pedestrian | 行人/非机动车安全预警 |
| **V2N** | Vehicle to Network | 高清地图、远程监控、云平台调度 |

**C-V2X（Cellular V2X）**：3GPP 标准化的蜂窝车联网技术，包含 **LTE-V2X（PC5 + Uu）** 与 **NR-V2X** 演进；**不等于**仅 NR，而是**蜂窝 + 直连**融合路线。

### 二、LTE-V2X 与 NR-V2X 演进关系

- **LTE-V2X**：以 LTE 物理层为基础定义 **PC5 直连**，成熟早，侧重安全广播类业务（如 BSM 等消息）。
- **NR-V2X（R16 起）**：更高吞吐、更灵活资源分配、更低时延潜力，支持**高级驾驶**、**协作感知**等；与 5G **uRLLC / eMBB** 能力协同。
- **二者关系**：**并存演进**，路侧与终端可长期 **LTE-V2X + NR-V2X** 多模；题库常见判断：**C-V2X 只能 NR 实现** 为 **错误**。

### 三、PC5 接口与 Uu 接口

| 接口 | 名称含义 | 典型用途 |
|------|----------|----------|
| **PC5** | 直连接口（Sidelink） | **车-车、车-RSU（UE 型）、车-人**等近距直连，低时延广播/组播 |
| **Uu** | 蜂窝 Uu | **车-基站** 蜂窝链路，回传、广域调度、业务锚点 |

**易错**：**车与基站之间的蜂窝链路是 Uu**；**车与 RSU/他车直连** 多为 **PC5**。若题干说“车与基站直连接口是 PC5”，一般为 **错误**。

### 四、NR-V2X 资源分配：Mode 1 与 Mode 2

（术语与 LTE-V2X 中 Mode 3/Mode 4 思想延续，NR 侧由 gNB 配置与 RRC/信令细节定义。）

| 模式 | 调度主体 | 特点 |
|------|----------|------|
| **Mode 1** | **基站（gNB）集中调度**分配 sidelink 资源 | 资源可预期、易管理，覆盖好区域更可靠 |
| **Mode 2** | **UE 自主**资源选择（含感知 **Sensing**） | 无基站或弱覆盖时仍可持续通信，依赖终端算法与拥塞控制 |

**LTE-V2X 对照记忆**：基站集中调度对应 **Mode 3**；终端自主对应 **Mode 4**（题库常考）。

### 五、Sidelink 与 C-V2X 协议栈（备考层级）

- **Sidelink**：在 **PC5** 上的直连传输，含控制面与用户面；NR-V2X 在物理层、MAC 等增强以支持更灵活调度与反馈机制。
- **C-V2X 协议栈**（概念上）：**PHY → MAC → RLC → PDCP**；用户面 **PC5-U** 与 LTE 侧对比是真题常考点（**哪一层不存在**之类，见题库）。

### 六、频段与产业部署（中国）

- 工信部已规划车联网**直连通信**频段；备考与题库常见口径：**5905～5925 MHz**（具体以最新法规与题库题干为准）。
- **RSU（Road Side Unit）**：路侧单元，常与 **OBU（On-Board Unit）** 通过 **PC5** 交互。

### 七、应用场景

- **辅助驾驶**：前方事故、闯红灯预警、交叉路口碰撞预警。
- **自动驾驶 / 协同**：传感器共享、协作变道、车队编队（低时延 + 高可靠）。
- **远程驾驶**：5G **Uu 大上行 + 低时延** 回传视频与控制。
- **智慧交通**：MEC 与 V2N 结合，区域交通优化。

## 考点速记

| 考点 | 记忆要点 |
|------|----------|
| V2X 四类 | **V2V / V2I / V2P / V2N** |
| C-V2X | 蜂窝标准车联网，**含 LTE 与 NR** |
| PC5 | **直连**（车车、车与 RSU 等） |
| Uu | **车与基站**蜂窝 |
| NR Mode 1/2 | **基站调度** / **UE 自主（sensing）** |
| LTE 对照 | Mode **3** 基站调度，Mode **4** 终端自主 |
| 中国直连频段 | **5905～5925 MHz**（题库高频） |
| RSU / OBU | 路侧 / 车载；与 PC5 强相关 |
| 判断题陷阱 | C-V2X **并非**只有 NR；车-基站 **不是** PC5 |
