# 仿真实验05：工程实践 - 5G网络开通调测

> 参考视频：充电时光、不想做仿真的熊二、-70L 工程实践系列

## 实验概述

本实验是仿真中最核心的题目之一，考查5G基站从勘站规划到业务验证的完整流程。**省赛大概率考原题**，练熟即可拿满分。

## 完整流程

```
竞赛背景 → 勘站规划 → 网络部署 → 开通调测 → 业务验证
```

## 核心知识点与操作步骤

### 一、勘站规划

#### 射频参数配置

| 参数 | 配置方法 |
|------|----------|
| 方位角 | 三扇区标准：**0°/120°/240°**，从正北顺时针 |
| 天线挂高 | 根据题目数据相加 |
| 下倾角 | 电子下倾角 + 机械下倾角 |

#### 设备选型

| 场景 | AAU | MIMO |
|------|-----|------|
| 城区宏站 | 64T64R | Massive MIMO |
| 郊区 | 32T32R | Massive MIMO |
| 室内 | 4T4R/8T8R | 普通MIMO |

#### 光纤光模块选型

- 光纤：查IR接口类型，N78/N79选**1310单模**，4×2LC黑色
- 光模块：选**25G-1310**
- 电源：直流**DC -48V**

### 二、网络部署

#### 设备组成
```
DTTP-6216(BBU) + AAU + OMC机柜 + 核心网 + 业务服务器
```

#### 部署流程
```
核心网部署 → OMC部署 → BBU部署(机房) → AAU部署(铁塔) → GPS天线 → 开通调测
```

#### IP地址配置

- 各设备IP必须在同一网段或正确配置路由
- 注意子网掩码的正确性
- 必须配置三条路由：
  - 基站 → **AMF**（控制面信令）
  - 基站 → **UPF**（用户面数据）
  - 基站 → **OMC**（网管管理）

### 三、开通调测

1. 检查板卡状态（HBPOD/HBPOF）
2. 检查光模块方向（勿插反）
3. 检查DCPD供电开关（必须闭合）
4. 配置小区参数：
   - PCI（物理小区ID）
   - 频点（NR-ARFCN）
   - 带宽
   - SCS（子载波间距）

### 四、业务验证

1. UE附着测试
2. Ping连通性测试
3. 速率测试

## 仿真操作注意事项

1. 使用**谷歌浏览器**
2. 选项顺序可能变化，**看内容不看位置**
3. IP地址配置注意**子网掩码正确性**
4. 提前登录测试环境
5. 队友分工：每人主攻3-4个仿真题
