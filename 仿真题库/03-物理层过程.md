# 仿真实验03：物理层过程

> 参考视频：不想做仿真的熊二、J_K_Wang、-70L 物理层系列

## 实验概述

本实验考查5G NR物理层的帧结构、资源映射、信道与信号等核心内容。是仿真中技术含量较高的题目。

## 核心知识点

### 帧结构

- 1帧 = 10ms = 10子帧
- 1子帧 = 1ms
- Normal CP: **14**符号/时隙，Extended CP: **12**符号/时隙
- SCS = 15×2^μ kHz，每子帧时隙数 = 2^μ
- **子帧时隙数不可能=3**
- **60kHz**同时支持Normal和Extended CP
- SFN循环数 = **1024**

### 频段划分

| 频段 | 频率范围 | 最大带宽 |
|------|---------|---------|
| FR1 | <7.125GHz | 100MHz |
| FR2 | 24.25-52.6GHz | 400MHz |

### 时隙配比

- 3.5GHz SCS=30kHz配比：**DDDSUDDSUU**
- 2.6GHz配比：DDDDDDDSUU，S=6:4:4

### SSB同步信号块

- SSB = PSS + SSS + PBCH（CRS不属于SSB）
- 频域：20RB = **240**子载波
- 时域：**4**个OFDM符号
- PCI = 3×N_ID_1 + N_ID_2，共**1008**个

### 物理信道

| 信道 | 方向 | 调制/说明 |
|------|------|----------|
| PDCCH | 下行控制 | 调制=**QPSK** |
| PDSCH | 下行数据 | R15最大**256QAM** |
| PRACH | 上行随机接入 | Format0-3 ZC长度=**839**，仅**开环**功控 |

- 1 CCE = **6** REG
- CORESET时域最多**3**个符号
- CCE聚合等级：1/2/4/8/16（**无3**）
- PRACH FDM最大 = **8**
- PCFICH是LTE信道，不是5G

### 编码

- **Polar码** → 控制信道
- **LDPC码** → 数据信道

### 参考信号

- SRS = 上行探测参考信号
- 空闲态RSRP通过**PBCH DMRS**测量
- PBCH DMRS频域间隔**4**子载波

## 仿真操作要点

1. **帧时隙设计**：根据给定SCS计算时隙数、符号数
2. **SSB时频资源设计**：在时频网格上标注SSB位置
3. **下行参考信号设计**：DMRS映射位置
4. **时隙配比图设计**：按给定配比填写D/U/S及S中的分配
