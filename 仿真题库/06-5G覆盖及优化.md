# 仿真实验06：5G覆盖及优化

> 参考视频：不想做仿真的熊二《5G覆盖及优化》

## 实验概述

本实验考查5G网络覆盖规划与优化方法，包括天线参数调整、链路预算、覆盖分析等。

## 核心知识点

### 一、5G覆盖规划

#### 天线参数

| 参数 | 说明 | 调整原则 |
|------|------|----------|
| **方位角** | 天线指向（从正北顺时针） | 主瓣朝向用户密集区，同站两扇区≥90° |
| **下倾角** | 天线向下倾斜角度 | 过覆盖→增大；弱覆盖→减小 |
| **天线挂高** | 天线安装高度 | 密集城区25-35m，一般30-45m，郊区35-55m |

#### 方位角规划原则

- 同站两扇区夹角 ≥ **90°**
- 天线主瓣**朝向用户密集区域**
- 天线主瓣**不要朝向湖面/江面**（会产生反射干扰）
- ~~天线主瓣和道路夹角越小越好~~ ✗

#### 下倾角调整

| 场景 | 操作 | 效果 |
|------|------|------|
| **过覆盖** | 增大下倾角 | 缩小覆盖，抑制干扰 |
| **灯下黑** | 增大下倾角 | 改善近端覆盖 |
| **弱覆盖** | 减小下倾角 | 扩大覆盖范围 |

### 二、链路预算

#### 基本公式

```
最大允许路径损耗 = 发射功率 + 天线增益 - 馈线损耗 - 接收灵敏度 - 干扰余量 - 阴影衰落余量 - 穿透损耗
```

#### 5G覆盖半径影响因素

- 频段：频率越高，自由空间损耗越大，覆盖越小
- 发射功率和天线增益
- 场景（密集城区/城区/郊区/农村）
- 建筑穿透损耗

### 三、Massive MIMO

- 5G部署Massive MIMO主要解决：大容量问题、深度覆盖问题、天面不足问题
- **不是**解决穿透损耗问题
- 700MHz组网最大天线端口数：**4T4R**

### 四、PCI规划

```
PCI = 3 × N_ID_1 + N_ID_2
范围：0-1007
```

规划原则：
- 同站三扇区 **mod3不同**
- 相邻基站PCI不同
- mod30尽量不同（避免DMRS冲突）

### 五、频率规划

NR-ARFCN计算（3000-24250MHz）：
```
NR-ARFCN = (频率MHz - 3000) / 0.015 + 600000
```

FR1信道栅格包括：15kHz、30kHz、100kHz（**不含5kHz**）

## 仿真操作要点

1. 根据题目场景选择合适的天线参数
2. 正确计算方位角（三扇区120°间隔）
3. 根据覆盖需求调整下倾角
4. PCI规划确保满足mod3规则
5. 理解链路预算各参数含义
