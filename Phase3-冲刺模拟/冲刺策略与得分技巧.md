### 保底得分计算 [CSDN-国一回忆录]

```
仿真：练熟原题 → 200分（大概率满分）

理论保底（完全不会时的蒙题下限）：
  单选蒙对率25% × 30题 × 3分 = 22.5分
  判断蒙对率50% × 10题 × 3分 = 15分
  多选（保守估计）≈ 12分
  合计 ≈ 50分

认真学习后理论预估：
  单选正确率70% × 30 × 3 = 63分
  多选正确率50% × 20 × 6 = 60分
  判断正确率80% × 10 × 3 = 24分
  合计 ≈ 147分

目标总分 = 200(仿真) + 147(理论) = 347分 → 稳拿省奖
```

> **核心结论**：仿真200分是保底，理论只要不交白卷就能拿奖 [CSDN-国一回忆录]

## 三、理论答题策略

### 单选题（90分 = 30题×3分）

**目标正确率：70%以上**

策略：
1. **排除法**：先排明显错误选项
2. **关键词匹配**：抓题目关键词→匹配知识点
3. **时间控制**：每题30-40秒
4. 不会的标记跳过，最后回看

### 多选题（120分 = 20题×6分）— 拉分关键

**目标正确率：50%以上**

> **错选漏选都不得分！**

策略：
1. **确定性原则**：只选确定正确的，不确定的**不选**
2. **全选型识别**：真题中约30%多选题答案是**全选ABCD**
3. **常见全选题型**：
   - "以下关于XX描述正确的是"→ 常常全选
   - "以下属于XX的选项是"→ 常常全选
   - "XX包括哪些"→ 常常全选
4. **时间控制**：每题45-60秒

### 判断题（30分 = 10题×3分）

**目标正确率：80%以上**

策略：
1. **绝对化陷阱**：含"只有""所有""一定"的多为错
2. **细节篡改**：改一个数值或名词让对变错
3. **时间控制**：每题15-20秒

### 答题顺序

```
推荐：判断(快速) → 单选(中等) → 多选(最难)
```

## 四、仿真考试策略

> **仿真大概率考原题** → 练到秒选 [CSDN-国一回忆录]

| 队友 | 负责题目 |

| A | 网络开通调测 + 信令流程 + 射频参数 |

### 时间分配

| 阶段 | 时间 |
|------|------|
| 审题 | 2分钟 |
| 各自操作 | 20-30分钟 |
| 交叉检查 | 5-8分钟 |

1. 使用**谷歌浏览器**
2. 选项顺序可能变化 → **看内容不看位置**
3. IP配置注意子网掩码
4. 先做会的题
