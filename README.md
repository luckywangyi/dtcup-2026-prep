## 赛事变化提醒

第十三届赛事已更名为**"大学生新一代信息通信科技大赛"**（原"大唐杯"），简称**信科赛**。

**2026年大纲新增重点**：
- 卫星通信/星地融合（NTN）、通感一体化（ISAC）、C-V2X车路协同、通算一体化
- 5G-A/6G技术演进（R18+）
- 经济决策分析（作业变动成本法、盈亏临界点）要求提升至"掌握"

## 使用指南

### 复习路径建议

1. **先看大纲对照目录**：`大纲对照-本科A组/` 严格按官方大纲组织，每个文档标注了考核要求级别（精通/掌握/熟悉）、核心知识点、考点速记表、相关真题、参考资源
2. **精通级优先**：1.1-基本原理的8个主题是最高优先级（40%比重中的核心）
3. **结合仿真题库**：仿真赛200分是保底分数，7道题反复练习
4. **查阅课程笔记**：`课程笔记/` 保留了学唐平台原始课程对应的详细笔记，可作为深入参考
5. **真题刷题**：`真题题库/` 中按知识点分类的1006道真题是最佳练习资源

### 大纲对照文档说明

每篇文档统一包含：
- **大纲分类** + **考核要求级别**
- **核心知识点**（详细技术内容）
- **考点速记表**（快速复习用）
- **相关真题**（从真题库中匹配的历年考题）
- **参考资源**（在线学习链接）
