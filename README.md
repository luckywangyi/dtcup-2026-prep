## 考试结构

| 部分 | 分值 | 说明 |
|------|------|------|
| 理论赛 | 300分 | 30单选(3分)+20多选(6分)+10判断(3分) |
| 仿真赛 | 200分 | 虚拟仿真操作（大概率原题） |
| **总分** | **500分** | 两人一组，理论各答取均值 |

## 核心策略

1. **仿真200分保底**：练到秒选，拿满分
2. **理论聚焦高频**：背诵96组重复真题 + 攻克321道多选
3. **每日6小时**：上午仿真 + 下午理论 + 晚上刷题
