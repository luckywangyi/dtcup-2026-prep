## 赛事变化提醒

第十三届赛事已更名为**"大学生新一代信息通信科技大赛"**（原"大唐杯"），简称**信科赛**。

**2026年大纲新增重点**：
- 卫星通信（3门课程）：网络架构、产品分类、空口技术
- 通感一体化（ISAC）：5G-A核心技术
- 5G网络节能技术及算法
- 这些内容在往年真题中占比很低，但今年是学唐平台的必修课，预计大幅增加出题

## 考试结构

| 部分 | 分值 | 说明 |
|------|------|------|
| 理论赛 | 300分 | 30单选(5分)+20多选(6分)+10判断(3分) |
| 仿真赛 | 200分 | 虚拟仿真操作（大概率原题） |
| **总分** | **500分** | 两人一组，理论各答取均值 |

## 核心策略

1. **仿真200分保底**：练到秒选，7道仿真题反复练习拿满分
2. **新增内容重点攻克**：卫星3门课+通感技术是今年新增必考
3. **传统高频不放松**：协议信令+基站产品是每年高频考点
4. **多选题攻克**：历年321道多选题重点辨析
